PRIMEIROS COMANDOS.
git init - iniciar um repositorio
git add serve para adicionar o projeto a um stage
git commit -m serve para mandar o arquivo para a rede

configurar email: git config --global user.email "danrocsan@gmail.com"
configurando usuario: git config --global user.name danrocsan


git config --list -> acessar toda a configuracao do git

git config --global --unset user.email -> tirar o email

git config --global --unset user.nickname  -> tirar o usuario
